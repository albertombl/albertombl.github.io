Crear un botón que ejecute nuestros programas

Resulta un incordio que nuestros script se ejecuten siempre al cargar la página. En este ejemplo vamos a crear un botón en nuestra página que ejecute un programa en Javascript

Los botones los podemos incluir en nuestros archivos html usando el siguiente código código

<input class="la_clase_que_desees" type="button" value="texto_en_botón" onclick="función_que_queremos_ejecutar" />

Es importantísimo entender para qué sirve cada uno de los atributos de esta etiqueta:

  • class="la_clase_que_desees" A estas alturas no hace falta explicar que a través de CSS podemos darle estilo a cualquier elemento de HTML y este no iba a ser la excepción
  • type="button" Al ser tipo button (botón) hereda características clave de los botones de HTML. Es decir, es lo que nos genera la forma de un botón en el que podemos hacer click
  • value="texto_en_botón" Este es el texto que saldría en el interior del botón
  • IMPORTANTE:
  • onclick="función_que_queremos_ejecutar" Habiendo elegido el archivo donde se encuentran el script (programa) que incluye las funciones (pequeños scripts o pequeños programas) solo nos queda elegir cuál ha de ejecutarse al hacer click en eñ botón. Este atributo nos permite elegir esa función.
  • NOTA IMPORTANTE: Como ves, en este caso el nombre del archivo es el mismo que el nombre de la función ejecutada. Esto ocurrirá siempre que queramos ejecutar todo el archivo, es decir, TODAS LAS FUNCIONES QUE INCORPORE ESTE.

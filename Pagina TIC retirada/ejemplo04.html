FUNCIONES

En este punto es interesante reflexionar sobre lo que es un programa. Pregunta a tu profesor. Normalmente usamos indistintamente las palabras programa, script, función, método... Esto se debe a que, en esencia, todos ellos son programas (secuencia algorítmica de instrucciones), pero cada palabra agrega un matiz concreto en base al contexto.

Hasta ahora, hemos construido dos script (programas) diferentes: alerta.js y pide_nombre.js .Estos debían ejecutarse completos cuando eran llamados. Al crear los botones, especificamos en el atributo correspondiente onclick="función_que_queremos_ejecutar" sus nombres como si fueran funciones de la siguiente manera:

<input class="clickMe" type="button" value="pide_nombre.js" onclick="pide_nombre()" /> <input class="clickMe" type="button" value="alerta.js" onclick="alerta()" />

Ahora bien, cuando tenemos un gran número de pequeños script/programas, nos es útil tenerlos todos en un mismo archivo identificándolos como funciones. Es decir, ya vemos que las funciones son programas que están escritos dentro de otros programas....

La forma en la que definimos una función en Javascript es la siguiente: 


    function pide_nombre()
    {
      //Dentro de estas llaves irá el programa pide_nombre
      var nombre;
      nombre=prompt("Escribe tu nombre","Nombre")
      alert("Hola " + nombre + ".\nEspero que veas el potencial de esta herramienta.\n¿Sabes quien soy?");
      alert("Soy TICO y tu profesor me ha programado.");
    }

Pues bien, vamos a crear un archivo llamado programas.js en el que incluiréis la función alerta y la función pide_nombre, después generaremos una nueva página web con unos botones que llamarán a estas funciones..

Ejercicio 4. Creación de funciones

  1. Crea un archivo javascript llamado programas.js
  2. Declara las funciones alerta y pide_nombre, tal y como se ha explicado antes dentro del archivo programas
  3. Ahora crea una nueva página web que deberás linkar con un enlace a la que ya teníamos.
  4. Enlaza el script programas.js a la página
  5. Incorpora dos botones, uno para cada función alerta.js y pide_nombre.js
  6. Finalmente, sube los archivos al campus virtual en una carpeta comprimida llamada tu_nombre_ejercicio_4.zip
